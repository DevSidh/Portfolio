Why I'm doing this

I started with Python and web fundamentals — HTML, CSS and JS — then went lower-level with C and C++ to actually understand what's happening under the hood. Lately I've been dabbling in PyTorch, poking at how models are built rather than just using them.

Alongside CSIT coursework, I'm also preparing for CEE, Nepal's medical entrance exam — two very different tracks running at the same time, on purpose.
